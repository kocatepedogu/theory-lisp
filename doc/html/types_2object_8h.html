◆ object_vtable_t

typedef struct object_vtable object_vtable_t

object_vtable_t contains operations that are supported by an object.

All objects must implement destroy, clone, tostring and equals functions. The remaining are optional. If an operation is not implemented, the member corresponding to it must be set to NULL. If an unsupported operation is called, it will return an error object containing "unsupported operation" message.
